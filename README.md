# Matematik 3c - Kurshemsida

En modern och interaktiv hemsida för att lära sig Matematik 3c.

## Funktioner

### ✅ Genomfört
- **Interaktiva grafer** med Plotly.js
  - Sekanter ritas upp automatiskt
  - Punkter markeras tydligt
  - Kurvor visas i olika färger

- **Färgade informationsrutor**
  - Blå för definitioner
  - Lila för viktig info
  - Orange för formler
  - Röd för extra viktiga formler
  - Grön för exempel
  - Och många fler!

- **LaTeX-stöd** via MathJax
  - Alla matematiska uttryck renderas snyggt

- **Dropdown-navigation**
  - Alla 6 kapitel med underavsnitt
  - Sticky navigation som följer med

- **Responsiv design**
  - Fungerar på alla enheter

## Interaktiva grafer som finns just nu

1. **Sekant på en kurva** - INTERAKTIV
   - Två slidebars för att flytta punkterna x₁ och x₂
   - Sekanten uppdateras automatiskt
   - Visar lutningen k i realtid

2. **Granplantans tillväxt** - INTERAKTIV
   - En slidebar för att välja mättidpunkten (år 0.5 till 3.5)
   - Visar genomsnittlig tillväxthastighet i dm/år
   - Praktiskt exempel på medellutning

3. **Ändringskvot för f(x) = x²** - INTERAKTIV
   - Två slidebars för att välja x₁ och x₂
   - Visar hela beräkningen av k = Δy/Δx
   - Experimentera och se hur lutningen förändras!

## Hur man använder

1. Öppna `index.html` i din webbläsare
2. Navigera med dropdown-menyerna
3. Interagera med graferna (zooma, panorera, hovra över punkter)

## Nästa steg

- Lägg till fler PDF-genomgångar
- Rita tangenter för derivata-avsnitten
- Lägg till grafer för integraler (arean under kurvor)
- Rita trigonometriska funktioner och enhetscirkeln
- Lägg till interaktiva övningar

## Teknisk stack

- **HTML5** för struktur
- **CSS3** för design (gradients, flexbox, responsive)
- **JavaScript** för interaktivitet
- **MathJax 3** för LaTeX-rendering
- **Plotly.js** för interaktiva grafer
