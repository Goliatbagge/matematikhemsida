Kapitel 5.6: Area mellan kurvor

Area mellan två kurvor

Arean mellan $y = f(x)$ och $y = g(x)$ från $x = a$ till $x = b$ (där $f(x) \geq g(x)$):

$$A = \int_a^b [f(x) - g(x)] \, dx$$

Area mellan två kurvor: A = ∫ₐᵇ [f(x) - g(x)] dx

Exempel 1: Area mellan två kurvor

Beräkna arean mellan kurvorna $y = x^2$ och $y = x$ från $x = 0$ till $x = 1$.

Steg 1: Bestäm vilken kurva som ligger högst

För $0 \leq x \leq 1$ gäller $x \geq x^2$ (testa t.ex. $x = 0{,}5$)

Steg 2: Beräkna integralen

$$A = \int_0^1 (x - x^2) \, dx = \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1$$

$$= \frac{1}{2} - \frac{1}{3} = \frac{3-2}{6} = \frac{1}{6}$$

Svar: $\frac{1}{6}$ ae

Exempel 2: Hitta skärningspunkter först

Beräkna arean mellan $f(x) = x^2 - 4$ och $g(x) = -x^2 + 2x + 4$.

Steg 1: Hitta skärningspunkter

$$x^2 - 4 = -x^2 + 2x + 4$$

$$2x^2 - 2x - 8 = 0$$

$$x^2 - x - 4 = 0$$

$$x = -2 \text{ eller } x = 4$$

Steg 2: Bestäm vilken kurva som ligger högst

$g(x) \geq f(x)$ mellan skärningspunkterna

Steg 3: Beräkna arean

$$A = \int_{-2}^4 [(-x^2 + 2x + 4) - (x^2 - 4)] \, dx$$

$$= \int_{-2}^4 (-2x^2 + 2x + 8) \, dx = \left[-\frac{2x^3}{3} + x^2 + 8x\right]_{-2}^4$$

$$= \left(-\frac{128}{3} + 16 + 32\right) - \left(\frac{16}{3} + 4 - 16\right) = 36$$

Svar: 36 ae
