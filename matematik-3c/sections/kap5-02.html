Kapitel 5.2: Primitiva funktioner med villkor

Figur 5.2: Bestämning av integrationskonstanten C med villkor

När vi vet funktionsvärdet i en viss punkt kan vi bestämma integrationskonstanten $C$.

Exempel 1: Bestäm C med villkor

Bestäm den primitiva funktionen till $f(x) = 2x + 3$ som går genom punkten $(1, 7)$.

Steg 1: Integrera

$$F(x) = \int (2x + 3) \, dx = x^2 + 3x + C$$

Steg 2: Använd villkoret $F(1) = 7$

$$1^2 + 3 \cdot 1 + C = 7$$

$$1 + 3 + C = 7$$

$$C = 3$$

Svar: $F(x) = x^2 + 3x + 3$

Exempel 2: Bestäm funktion från derivata

En funktion $f$ har derivatan $f'(x) = 6x - 4$ och $f(2) = 10$. Bestäm $f(x)$.

$$f(x) = \int (6x - 4) \, dx = 3x^2 - 4x + C$$

$$f(2) = 3 \cdot 4 - 8 + C = 10$$

$$12 - 8 + C = 10$$

$$C = 6$$

Svar: $f(x) = 3x^2 - 4x + 6$
