Kapitel 3.1: Derivatan av enkla potensfunktioner

För att slippa använda oss av derivatans definition när derivatan ska beräknas söker vi enkla deriveringsregler.

Kom ihåg: Derivatans definition

$$f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h}$$

Härledning av deriveringsregler

Derivatan för $f(x) = x$

$$f'(x) = \lim_{h \to 0} \frac{(x + h) - x}{h} = \lim_{h \to 0} \frac{h}{h} = \lim_{h \to 0} 1 = 1$$

Så $f(x) = x$ har derivatan $f'(x) = 1$

Derivatan för $f(x) = x^2$

$$f'(x) = \lim_{h \to 0} \frac{(x + h)^2 - x^2}{h} = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}$$

$$= \lim_{h \to 0} \frac{2xh + h^2}{h} = \lim_{h \to 0} \frac{h(2x + h)}{h} = \lim_{h \to 0} (2x + h) = 2x$$

Så $f(x) = x^2$ har derivatan $f'(x) = 2x$

Potensfunktioner och deras derivator: Derivatan har alltid en grad lägre

Mönster för derivatan

$f(x)$ $x$ $x^2$ $x^3$ $x^4$ $x^5$
$f'(x)$ $1$ $2x$ $3x^2$ $4x^3$ $5x^4$

Vi ser att derivatan följer ett mönster: När vi deriverar en potensfunktion "multipliceras exponenten ned och minskar sedan med 1".

Derivatan av $f(x) = x^n$

$$f(x) = x^n \quad \Rightarrow \quad f'(x) = nx^{n-1}$$

Derivatan av $f(x) = kx^n$

$$f(x) = kx^n \quad \Rightarrow \quad f'(x) = nkx^{n-1}$$

Derivatan av $f(x) = kx$

$$f(x) = kx \quad \Rightarrow \quad f'(x) = k$$

En följd av regeln är att när x-termer (av grad 1) deriveras, så "försvinner" x:et.

Derivatan av $f(x) = \frac{g(x)}{a}$

$$f(x) = \frac{g(x)}{a} \quad \Rightarrow \quad f'(x) = \frac{g'(x)}{a}$$

När vi deriverar funktioner med konstanter i nämnaren, deriveras täljaren medan nämnaren behålls.

Derivatan av $f(x) = a$ (konstant)

$$f(x) = a \quad \Rightarrow \quad f'(x) = 0$$

Derivatan av en konstant funktion är 0 (lutningen är 0 för alla konstanta funktioner).

Sammanfattning - Deriveringsregler

$f(x) = x^n$ $\Rightarrow$ $f'(x) = nx^{n-1}$
$f(x) = kx^n$ $\Rightarrow$ $f'(x) = nkx^{n-1}$
$f(x) = a$ (konstant) $\Rightarrow$ $f'(x) = 0$
$f(x) = kx$ $\Rightarrow$ $f'(x) = k$
$f(x) = \frac{g(x)}{a}$ $\Rightarrow$ $f'(x) = \frac{g'(x)}{a}$

Derivera och bestämma derivatans värde

Utifrån en funktion $f(x)$ bestämma derivatans värde i en bestämd punkt $f'(a)$:

  1. Bestäm $f'(x)$ med deriveringsregler.
  2. Sätt sedan in $x = a$ i derivatafunktionen och bestäm $f'(a)$.

Exempel 1: Derivera

a) $f(x) = x^7$

Vi multiplicerar ner exponenten 7 och minskar den sedan med 1:

$$f'(x) = 7x^6$$

Svar: $f'(x) = 7x^6$

b) $y = 3x^5$

Vi multiplicerar ner exponenten 5 och minskar den sedan med 1:

$$y' = 5 \cdot 3x^4 = 15x^4$$

Svar: $y' = 15x^4$

c) $f(x) = 27$

Derivatan till konstanta funktioner är 0:

$$f'(x) = 0$$

Svar: $f'(x) = 0$

d) $g(x) = 12x$

När vi deriverar x-termer "försvinner" x:et:

$$g'(x) = 12$$

Svar: $g'(x) = 12$

e) $f(x) = \frac{x^4}{5}$

Vi deriverar täljaren och behåller nämnaren:

$$f'(x) = \frac{4x^3}{5}$$

Svar: $f'(x) = \frac{4x^3}{5}$

Exempel 2: Bestäm $f'(3)$ för $f(x) = 4x^2$

Vi bestämmer först $f'(x)$ med deriveringsreglerna och sedan sätter vi in $x = 3$:

$$f'(x) = 8x$$

$$f'(3) = 8 \cdot 3 = 24$$

Svar: $f'(3) = 24$
